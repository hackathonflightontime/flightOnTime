# Notebooks — Data Science

Este diretório contém os notebooks utilizados durante o desenvolvimento
do modelo preditivo do projeto FlightOnTime.

## EDA_Voo.ipynb
Responsável pela análise exploratória dos dados:
- verificação de integridade
- distribuição do alvo
- entendimento do perfil dos voos

## Feature_Engineering.ipynb
Responsável pela criação das variáveis utilizadas pelo modelo:
- extração de mês e dia da semana
- codificação de variáveis categóricas
- preparação final do dataset

## Treinamento_Modelo.ipynb
Responsável pelo treinamento, avaliação e geração dos artefatos finais:
- divisão treino/teste
- treinamento do Random Forest
- avaliação do modelo
- exportação do modelo e do mapeamento
